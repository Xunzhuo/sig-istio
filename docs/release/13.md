# Istio 1.3——改善用户体验

2 个月前发布的 0.8 版本已经为 1.0 的发布做好大量的前置工作，经过这 2 个月的时间，0.8 版本的众多 alpha 功能现在变成了 beta，新增了支持将多个 Kubernetes 集群添加到单个 mesh 集群中。

在此之前社区对于 Istio 的性能质疑声不断，本版本对 Istio 的性能做了大量优化，虽然号称生产就绪，但是此时还没有充足的生产案例。
