# Istio 0.5——对用户采用更友好

该版本主要增强易用性，相对于 Istio 初期一键安装所有组件的情况，现在 Istio 用户渐渐式采用，可以只安装 Istio 的部分组件。Istio 利用 Kubernetes 1.9 及以上版本的 muting webhook 特性，支持自动注入 sidecar。
